# cereallovers

-- backend ------------------------------------------------------------------
* a get route that fetch data of multiple users based on their cereals preference.
* a post route to add new users 


-- frontend-------------------------------------------------------------------
* linking pages from (/ - home - showCereals - ShowUsers - showCereals)
* a form to submit new user details
* a list of cereals with a check box to be checked individually 
* update the checked cereals and render the users who are interested in the same cereals
* click each photo of the users icon to display on the top of the page 

----------------------------------------------------------------------------------------
* a radio button to select their gender and gender in interest and display the users profile based on that.
