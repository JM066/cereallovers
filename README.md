# cereallovers

-- backend ------------------------------------------------------------------
get routers that fetch users data based on their cereals and gender preference.

-- frontend-------------------------------------------------------------------
a list of cereals with a check box to be checked individually by clicking the image icon and update the state to be called later to display the users data (with profile) that belongs to each cereals type.
a radio button to select a gender in interest and display the users profile based on that.
a button click that renders users data obtained by clikcing favorite cereals and genders in interest.
